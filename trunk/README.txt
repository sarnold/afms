AFMS is an "Artefact Management System" to manage some of the artefacts  
appearing in the software development process of a product.
View the provided documentation for details.


REQUIREMENTS
============
* Python 2.5 or later must already be installed. 
  The latest release is recommended. Python is available from http://www.python.org/.
* wxPython 2.8.7.1, see http://www.wxpython.org/
* docutils 0.4, see http://docutils.sourceforge.net/


INSTALLATION
============
1. Expand the .tgz or .zip archive in an empty directory of your choice
2. For the artefact editor run afeditor.py
3. For test execution run testrunner.py
